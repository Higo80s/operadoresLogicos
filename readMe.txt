Calculadora Aritmética
Este é um projeto simples de uma calculadora desenvolvida com HTML, CSS e JavaScript que demonstra o uso de operadores aritméticos para realizar operações matemáticas básicas, como adição, subtração, multiplicação e divisão. A calculadora permite que os usuários insiram dois números e escolham a operação desejada para obter o resultado imediatamente.

Funcionalidades
A calculadora aritmética inclui as seguintes funcionalidades:

Entrada de Números: Os usuários podem inserir dois números em caixas de entrada para serem usados em operações matemáticas.

Seleção da Operação: Os usuários podem escolher a operação desejada (adição, subtração, multiplicação ou divisão) a partir de um menu suspenso.

Cálculos Instantâneos: Ao clicar no botão "Calcular", a calculadora realiza a operação matemática selecionada e exibe o resultado na página.

Lidando com a Divisão por Zero: A calculadora verifica se a divisão por zero é evitada e exibe uma mensagem de erro apropriada se necessário.

Como Usar
Para utilizar a calculadora aritmética, siga estas etapas:

Abra o arquivo index.html em um navegador da web.

Insira dois números nos campos de entrada designados.

Selecione a operação desejada no menu suspenso.

Clique no botão "Calcular" para obter o resultado da operação.

O resultado será exibido na página abaixo dos campos de entrada.
